# Reaction Recipients
Flask app that uses a Discord Chat Exporter to find and display every message with relevant reactions.

Run using:
```
python3 -m flask run
```
and follow terminal instructions to initialise the app with data. Content directories for the JSON files can either go in `static/` or Download assets be disabled for exports.

>[!IMPORTANT]
>Assumes a single user only on the Flask instance (localhost for **one** user viewing on their *own* machine).
